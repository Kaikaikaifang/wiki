## 定义

本地优先搜索指：索引、查询、嵌入生成与重排尽可能在本机完成，原始文档停留在本地文件系统中，不依赖把资料上传到外部服务。

## 价值

- **隐私**：个人笔记、会议记录、日记类资料不必离开本机
- **可迁移**：知识仍保留为普通 markdown 文件，而不是封进专有数据库
- **低耦合**：搜索层可替换，原始资料与 wiki 结构不受影响
- **稳定性**：不依赖在线 API 的限额、延迟和价格波动

## 在个人 wiki 里的位置

对于小型 wiki，`wiki/index.md` 已能承担导航功能；当资料数量增长后，本地搜索成为必要的第二层基础设施。它不替代人工策展或 LLM 综述，而是帮助系统在更大规模下保持可访问性。

## 典型实现

[[entities/qmd]] 是一个代表性实现：保留 markdown 文件为事实来源，在本机完成 BM25、向量检索与 LLM 重排，并为 agent 提供结构化输出。
